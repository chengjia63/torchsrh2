### Consensus Clustering
Created:
```text
ts2/utils/silica_eval/clustering/fit_consensus.py
```
It is analysis-first and does not implement inference yet.
Features:
- Global sampling via `global_sample_n`.
- Base clustering ensemble over multiple `k_range` values and random seeds.
- Defaults to `MiniBatchKMeans` because it should scale better for 2M normalized DINO-style SRH cell embeddings.
- Also supports GMM base clusterings with `algorithm="gmm"`.
- Produces:
  - `consensus_assignments.csv`
  - `consensus_assignments.npz`
  - `base_clustering_metrics.csv`
  - `base_clustering_specs.json`
  - `consensus_metrics.csv`
  - `consensus_cluster_summary.csv`
  - `cluster_similarity.csv`
  - `cluster_similarity.png/html`
  - sampled cell-cell co-association matrix artifacts
  - consensus TSNE artifacts
Outliers are reported with `is_outlier`; they are not removed automatically.
